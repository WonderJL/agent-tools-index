# Security

This repository is intended to contain only redacted metadata and hand-written narrative — no
source, credentials, or private identifiers. The publisher runs a fail-closed leak scan before any
content is pushed here.

If you believe a private path, credential, internal identifier, or other sensitive datum has
slipped through into this public repository, please open an issue (without quoting the sensitive
value) or contact the owner via the profile at `github.com/WonderJL`. It will be scrubbed at the
source and republished.
