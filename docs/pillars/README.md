# Per-pillar narrative

A narrative page for each of the nine capability pillars in this index. Start with [Architecture](../architecture.md) for the whole-system map, or jump to a pillar:

- **[Skill](skill.md)** — procedural know-how, live + on-demand library
- **[Sub-agent](sub-agent.md)** — named personas dispatched for focused tasks
- **[CLI](cli.md)** — agent-facing command-line tools (AXI/TOON contract)
- **[Hook](hook.md)** — lifecycle automation, library-resolved and dormant-by-default
- **[Kit](kit.md)** — repo agentic-setup composed from hooks + skills
- **[Host](host.md)** — thin per-host adapters over a host-agnostic core
- **[Integration](integration.md)** — third-party tools vendored with provenance + license
- **[Schema](schema.md)** — JSON contracts validating manifests and kits
- **[ADR](adr.md)** — architecture decision records + design specs
